고블린 테마 몬스터 상세 명세서

현재 빌드 기준으로 고블린 테마 몬스터의 데이터 테이블 수치와 실제 전투 엔진에서 연결되는 행동 스킬을 정리한 문서다. 이 문서는 data.js의 몬스터 템플릿과 game-engine.js의 intent 처리 로직을 함께 읽은 결과를 하나의 기획/운영 문서로 묶는다.

최종 갱신: 2026-03-17 01:01 대상 테마: goblin 정상 몬스터 4종 + 보스 1종 + 최종보스 1종 현재 빌드 기준
중요: 현재 빌드는 고블린 몬스터용 별도 스킬 데이터 테이블을 따로 두지 않는다. 실제 연결 스킬은 intents[], element, 그리고 선택적으로 붙는 특수 필드가 합쳐져 결정된다. 즉 이 문서의 `연결 스킬`은 개별 스킬 asset이 아니라 `행동 intent + 상태 부여 + 공통 처리 로직` 해석값이다.
고블린 테마 정체성 템포 압박 + 붕괴 압박 + 자기 강화

슬라임보다 공격적이고, 리치보다 직선적이다. 방어/붕괴/격노를 섞어 플레이어에게 빠른 정리 압박을 건다.

속성 분포 무속성 / 냉기 / 전격 / 대지

고블린 대장은 대지 쇄약, 보물 고블린 왕은 전격 감전으로 후반 피해 증폭을 건다.

역할 커버리지 전열 / 방벽 / 돌격 / 보스

일반전에서는 전열과 돌격이, 변주 역할은 냉기 방벽과 전격 돌격이 맡는다.

1. 공통 해석 규칙

1.1 스폰 시 수치 계산식

항목현재 규칙
Normal HPbaseHp + floor((page - 1) / 2) * hpGrowth + grimoire.addMonsterHp에 페이지 구간 HP multiplier와 overrun multiplier를 곱한다.
Boss HPbaseHp + max(2, floor(page * 0.55)) * hpGrowth + grimoire.addMonsterHp 이후 boss multiplier 적용.
Final Boss HPbaseHp + max(4, floor(page * 0.65)) * hpGrowth + grimoire.addMonsterHp 이후 final-boss multiplier 적용.
Normal ATKbaseAttack + floor((page - 1) / 3) * attackGrowth + grimoire.addMonsterAttack에 구간 multiplier와 overrun multiplier를 곱한다.
Boss / Final ATKbaseAttack + floor(page / 2) * attackGrowth + grimoire.addMonsterAttack에 role multiplier와 overrun multiplier를 곱한다.
실전 공격력행동 시점의 실제 피해는 effectiveMonsterAttack을 사용한다. 여기엔 rage(+2/stack), monster chill(-2/stack), 일부 특수 패시브 보정이 추가된다.

1.2 intent 기반 행동 스킬

intent표시 라벨실제 효과
attack공격 N플레이어에게 effectiveMonsterAttack만큼 피해. 몬스터 속성이 있으면 원소 상태도 같이 부여.
defend / guard방어자신에게 Shield +6.
collapse붕괴 +7플레이어 붕괴 게이지 +7.
heal재생자신의 HP +16. 현재 HP가 35% 이하이고 heal intent가 있으면 우선 선택된다.
buff격노rage 1 / 3턴 부여. 실질 공격력 +2.
burn공격 N+2 + 화상공격력 +2 버전의 속성 공격. 현재 고블린 테마는 이 intent를 사용하지 않는다.

1.3 속성 연동 상태 이상

속성플레이어 피격 시 부여효과 요약
무속성없음추가 상태 없음.
냉기chill 1 / 2턴플레이어 기준 리롤 -1.
전격shock 1 / 2턴받는 피해 +20%.
대지fracture 1 / 2턴받는 피해 +10%, 턴 시작 Shield -2.

1.4 문서 해석 주의

  • 아래 표의 HP/ATK는 모두 템플릿의 기본값이다.
  • 실전 스폰 수치는 페이지, grimoire 보정, overrun 난이도에 따라 달라진다.
  • 고블린 테마는 별도 패시브 필드가 붙은 일반 몬스터가 아직 없다. 현재 연결 스킬은 전부 공통 intent 시스템 안에서 해석된다.
  • 보물 고블린 왕만 heal intent를 가져, 저체력 구간에서 행동 패턴이 바뀐다.

3. 몬스터별 연결 스킬 해석

고블린 스카웃 410003

무속성돌격

연결 스킬실제 효과
기본 공격무속성 공격. 추가 상태 이상 없음.
방어 태세Shield +6.

의도 가중치가 1:1이라 공격과 방어를 반반 섞는 초반 돌격형 기본 유닛이다.

노말 고블린 410004

무속성전열

연결 스킬실제 효과
기본 공격무속성 공격. 추가 상태 이상 없음.
방어 태세Shield +6.

스카웃보다 HP와 공격 성장치가 높아 전열 유지력이 더 좋다. 고블린 테마의 표준 기준점 역할을 한다.

얼음 고블린 410005

냉기방벽

연결 스킬실제 효과
냉기 공격공격 후 chill 1 / 2턴 부여. 플레이어 리롤 -1.
방어 태세Shield +6.
붕괴 유도플레이어 붕괴 게이지 +7.

리롤 압박과 방어/붕괴를 함께 가져, 고블린 테마 안에서 가장 `시간을 끄는` 변주 몬스터다.

전격 고블린 410007

전격돌격

연결 스킬실제 효과
전격 공격공격 후 shock 1 / 2턴 부여. 플레이어 받는 피해 +20%.
붕괴 유도플레이어 붕괴 게이지 +7.

방어 대신 감전과 붕괴를 고른 순수 압박형 고블린이다. 뒤에 다른 고블린이 있으면 누적 피해 기대값을 빠르게 올린다.

고블린 대장 410020

대지보스

연결 스킬실제 효과
대지 공격공격 후 fracture 1 / 2턴 부여. 받는 피해 +10%, 턴 시작 Shield -2.
격노rage 1 / 3턴. 실질 공격력 +2.
붕괴 유도플레이어 붕괴 게이지 +7.

쇄약으로 피해 증폭을 건 뒤 격노와 붕괴로 압박을 누적하는 보스다. 고블린 테마가 단순 다수전이 아니라 `중간 보스형 압박`도 가진다는 증거다.

보물 고블린 왕 410030

전격최종보스

연결 스킬실제 효과
전격 공격공격 후 shock 1 / 2턴 부여. 플레이어 받는 피해 +20%.
격노rage 1 / 3턴. 실질 공격력 +2.
붕괴 유도플레이어 붕괴 게이지 +7.
재생HP +16. HP가 35% 이하일 때는 heal intent를 우선 선택한다.

감전으로 피해 증폭을 깔고, 격노로 킬 턴을 압박하며, 저체력 구간에는 재생으로 전투를 끈다. 현재 고블린 최종보스는 `누적 압박 + 막판 연장전`의 성격이 강하다.

4. 기획 해석 메모

현재 고블린 테마가 잘하는 것

  • 무속성 기본형, 냉기 방벽형, 전격 돌격형으로 normal 변주 폭이 적절하다.
  • 보스는 대지 쇄약, 최종보스는 전격 감전으로 속성 압박 축이 분명하다.
  • defend/guard, collapse, buff가 섞이면서 `빨리 끝내야 하는 적`의 인상을 잘 만든다.

현재 비어 있는 부분

  • 리치 테마처럼 개별 패시브 필드가 붙은 고블린 일반 몬스터가 아직 없다.
  • 고블린 테마만의 고유 행위, 예를 들면 약탈, 골드 변환, 추가 소환, 보상 간섭은 몬스터 쪽에 아직 직접 붙어 있지 않다.
  • 즉 현재 고블린 몬스터 디자인은 `행동 템포`는 선명하지만 `경제/약탈 판타지`는 책과 유물 쪽에 더 많이 실려 있다.
